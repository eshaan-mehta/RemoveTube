Test Chrome Extension Storage

This page tests the API key persistence functionality of the RemoveTube extension.

Manual Test Instructions

  1. Load the RemoveTube extension in Chrome
  2. Visit YouTube.com
  3. The setup overlay should appear
  4. Test the API key collapsible functionality:
    • First time: API key input should be visible
    • Enter a test API key and save
    • Refresh YouTube - API key section should be collapsed with "Show/Edit Key" button
    • Click "Show/Edit Key" to expand the input
    • Click "Hide Key" to collapse it again
  5. Test that the API key persists across page reloads
